<h2><a href="https://leetcode.com/problems/middle-of-the-linked-list/">876. Middle of the Linked List</a></h2>
<h3>Easy</h3>
<hr>
<p>Given the <strong>head</strong> of a singly linked list, return <strong>the middle node</strong> of the linked list.</p>

<p>If there are two middle nodes, return <strong>the second</strong> middle node.</p>

<h3>Example 1:</h3>
<pre>
<b>Input:</b> head = [1,2,3,4,5]
<b>Output:</b> [3,4,5]
<b>Explanation:</b> The middle node is 3.
</pre>

<h3>Example 2:</h3>
<pre>
<b>Input:</b> head = [1,2,3,4,5,6]
<b>Output:</b> [4,5,6]
<b>Explanation:</b> The middle nodes are 3 and 4, so the second middle node is returned (node with value 4).
</pre>

<hr>
<h3>Constraints:</h3>
<ul>
  <li>The number of nodes in the list is in the range <code>[1, 100]</code>.</li>
  <li><code>1 &lt;= Node.val &lt;= 100</code></li>
</ul>
