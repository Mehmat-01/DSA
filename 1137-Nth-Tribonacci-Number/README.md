<h2><a href="https://leetcode.com/problems/n-th-tribonacci-number/">1137. N-th Tribonacci Number</a></h2>
<h3>Easy</h3>
<hr>
<p>The Tribonacci sequence T(n) is defined as follows:</p>

<pre>
T(0) = 0  
T(1) = 1  
T(2) = 1  
T(n) = T(n−1) + T(n−2) + T(n−3), for n >= 3
</pre>

<p>Given <code>n</code>, return the value of <code>T(n)</code>.</p>

<h3>Example 1:</h3>
<pre>
<b>Input:</b> n = 4
<b>Output:</b> 4
<b>Explanation:</b> T(4) = T(3) + T(2) + T(1) = 2 + 1 + 1 = 4.
</pre>

<h3>Example 2:</h3>
<pre>
<b>Input:</b> n = 25
<b>Output:</b> 1389537
</pre>

<hr>
<h3>Constraints:</h3>
<ul>
  <li><code>0 &lt;= n &lt;= 37</code></li>
</ul>
