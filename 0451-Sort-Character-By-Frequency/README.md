<h2><a href="https://leetcode.com/problems/sort-characters-by-frequency/">451. Sort Characters By Frequency</a></h2>
<h3>Medium</h3>
<hr>
<p>Given a string <code>s</code>, sort it in <strong>decreasing order</strong> based on the frequency of the characters, and return the sorted string.</p>

<h3>Example 1:</h3>
<pre>
<b>Input:</b> s = "tree"
<b>Output:</b> "eert"
<b>Explanation:</b> 'e' appears twice while 'r' and 't' appear once.
So "eetr" is also valid.
</pre>

<h3>Example 2:</h3>
<pre>
<b>Input:</b> s = "cccaaa"
<b>Output:</b> "aaaccc"
<b>Explanation:</b> Both 'c' and 'a' appear three times, so either "aaaccc" or "cccaaa" is valid.
</pre>

<h3>Example 3:</h3>
<pre>
<b>Input:</b> s = "Aabb"
<b>Output:</b> "bbAa"
<b>Explanation:</b> "bbaA" is also valid. Case is treated as distinct.
</pre>

<hr>
<h3>Constraints:</h3>
<ul>
  <li><code>1 &lt;= s.length &lt;= 5 * 10<sup>5</sup></code></li>
  <li><code>s</code> consists of uppercase and lowercase English letters and digits.</li>
</ul>
