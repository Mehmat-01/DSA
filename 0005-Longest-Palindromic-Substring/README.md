<h2><a href="https://leetcode.com/problems/longest-palindromic-substring/">5. Longest Palindromic Substring</a></h2>
<h3>Medium</h3>
<hr>
<p>Given a string <code>s</code>, return the <strong>longest palindromic substring</strong> in <code>s</code>.</p>

<h3>Example 1:</h3>
<pre>
<b>Input:</b> s = "babad"
<b>Output:</b> "bab"
<b>Explanation:</b> "aba" is also a valid answer.
</pre>

<h3>Example 2:</h3>
<pre>
<b>Input:</b> s = "cbbd"
<b>Output:</b> "bb"
</pre>

<hr>
<h3>Constraints:</h3>
<ul>
  <li><code>1 &lt;= s.length &lt;= 1000</code></li>
  <li><code>s</code> consist of only digits and English letters.</li>
</ul>
