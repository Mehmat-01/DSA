<h2><a href="https://leetcode.com/problems/reverse-linked-list/">206. Reverse Linked List</a></h2>
<h3>Easy</h3>
<hr>
<p>Given the head of a singly linked list, reverse the list and return the reversed list.</p>

<h3>Example 1:</h3>
<pre>
<b>Input:</b> head = [1,2,3,4,5]
<b>Output:</b> [5,4,3,2,1]
</pre>

<h3>Example 2:</h3>
<pre>
<b>Input:</b> head = [1,2]
<b>Output:</b> [2,1]
</pre>

<h3>Example 3:</h3>
<pre>
<b>Input:</b> head = []
<b>Output:</b> []
</pre>

<hr>
<h3>Constraints:</h3>
<ul>
  <li>The number of nodes in the list is in the range <code>[0, 5000]</code>.</li>
  <li><code>-5000 &lt;= Node.val &lt;= 5000</code></li>
</ul>

<hr>
<h3>Follow up:</h3>
<p>A recursive solution is also possible. Can you do it iteratively?</p>
